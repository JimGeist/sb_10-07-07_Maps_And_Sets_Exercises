10.07.07 - Maps and Sets Exercises

Quick Questions #1 - What does new Set([1, 1, 2, 2, 3, 4]) return?

It returns {1, 2, 3, 4} because sets cannot have duplicates.

Quick Question #2 - What does [...new Set("referee")].join("") return?

It returns "ref" because sets cannot have duplicates and the duplicate letters are removed from the string when set iterates over the string. join("") concatenates each element of the set together with no spaces between each element.

Quick Question #3 - What does the Map m look like after running the following code?

  let m = new Map();
  m.set([1, 2, 3], true);
  m.set([1, 2, 3], false);

m will look like {[1, 2, 3] => true, [1, 2, 3] => false} because an array is the key to the set. Each set method calls with an array creates a new entry because an array is a reference object and each array in the method call is a new reference. To have only one array [1, 2, 3] key, declare the array and then call the m.set() method:
  const arr123 = [1,2,3];
  m.set(arr123, true);
  m.set(arr123, false);
will yield {[1, 2, 3] => false}.
